# hyperterm-light

> [Atom One Light][one-light] theme for [HyperTerm][hyperterm]

![](screenshot.png)

## Install

Open your HyperTerm preferences and add `hyperterm-light` to plugin list:

```js
plugins: [
  'hyperterm-light'
],
```
